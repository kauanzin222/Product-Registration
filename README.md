# 📦 Cadastro de Produtos - Lógica de Front-End e Responsividade

![Status](https://img.shields.io/badge/Status-Finalizado-green)

Este repositório registra o desenvolvimento do Módulo 5 do **BootCamp DevJr**, focado na criação de uma interface funcional para o gerenciamento de produtos. O projeto foi um passo importante para consolidar o uso de **JavaScript** na manipulação de dados e garantir uma experiência de usuário fluida em qualquer dispositivo.

## 🚀 Principais Aprendizados

O desenvolvimento deste módulo permitiu aprofundar competências cruciais para a minha meta de ser um **Desenvolvedor Full-Stack**:

- ⚡ **Manipulação Dinâmica do DOM**: Aprendi a utilizar JavaScript para capturar dados de formulários e atualizar a interface em tempo real, proporcionando uma navegação mais ágil.
- 🎨 **Estilização com Bootstrap**: Aplicação do **Bootstrap** (meu padrão oficial de design) para construir layouts modernos e componentes responsivos de forma eficiente.
- 📱 **Responsividade (Mobile First)**: Domínio do sistema de grid para assegurar que a aplicação mantenha a usabilidade e a fidelidade visual tanto em desktops quanto em dispositivos móveis.
- 🛠️ **Lógica de Interface**: Implementação de fluxos simples de entrada e saída de dados, preparando a base para futuras integrações com APIs e bancos de dados.

## 🛠️ Tecnologias Utilizadas

- **Linguagem**: JavaScript (ES6+).
- **Estruturação**: HTML5.
- **Design/UI**: Bootstrap 5.
